> 구현한 기능 

- Add 버튼과 Enter Key로 habit을 추가할 수 있습니다.
- habit의 횟수를 증가하고 감소시킬 수 있으며 삭제도 가능합니다.
- Reset All 버튼으로 횟수를 초기화할 수 있습니다.
